NSDictionary(GCKAdditions) Category Reference

Declared in NSDictionary+GCKAdditions.h

Overview

A category that adds some convenience methods to NSDictionary for setting and safely looking up values of various types. These methods are particularly useful for getting and setting fields of JSON data objects.
